Cap. 3 — Tablas geográficas de referencia

Estas tablas proporcionan la base cartográfica y los identificadores geográficos que utilizan todos los demás componentes del sistema. La mayoría se cargaron manualmente a partir de fuentes cartográficas oficiales (IGN, GRAFCAN, INE).

Jerarquía territorial

canarias (1)
  └── islas (7)
        └── municipios (88)
              └── localidades (2.346)

Tabla: callejero_portales

Tabla: callejero_portales  |  Filas actuales: 526751

Callejero de Canarias a nivel de portal (número de calle), con geometría de punto, dirección normalizada y código postal. Se usa en P02 para la geocodificación fuzzy de establecimientos turísticos mediante similitud trigonométrica (pg_trgm).

Campos clave para la geocodificación: dir_normalizada (dirección estandarizada para la búsqueda fuzzy), municipio_id, localidad_id, geom.

Fuente: cartografía oficial (GRAFCAN/Cartociudad). Carga manual.

Tabla: destinos_turisticos

Tabla: destinos_turisticos  |  Filas actuales: 23

Los 23 microdestinos turísticos reglados de Canarias, delimitados por el planeamiento urbanístico de suelo de uso turístico. Se usan en P09 para clasificar cada alojamiento según si está en un área turística oficial (en_area_turistica = TRUE) o en suelo residencial.

El campo turistica distingue entre zonas con ordenación turística activa y las bolsas de suelo sin microdestino definido (que se tratarán como suelo residencial a efectos del análisis).

Fuente: delimitación oficial de zonas turísticas del GobCan / GRAFCAN. Carga manual.
