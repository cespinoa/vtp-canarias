Cap. 7 — Scripts de descarga y mantenimiento

Scripts Python para la descarga de datos desde las APIs públicas del GobCan, ISTAC e INE. Se encuentran en descarga_datos/. Cada script genera un fichero CSV en tmp/que el script R de importación correspondiente carga en la base de datos.

Registros turísticos (GobCan / CKAN)

importar_registro_alojamientos.py

Descarga los tres datasets de registros turísticos del GobCan desde la API CKAN. Usa el campo metadata_modified del dataset como fecha del fichero, lo que permite detectar si hay datos nuevos sin necesidad de comparar contenidos.

  • Viviendas vacacionales: → importar_gobcan/historico/vv-YYYY-MM-DD.csv
  • Establecimientos hoteleros: → importar_gobcan/historico/ht-YYYY-MM-DD.csv
  • Extrahoteleros sin VV: → importar_gobcan/historico/ap-YYYY-MM-DD.csv

No sobreescribe si ya existe un fichero con la misma fecha. Tras la descarga, ejecutar el pipeline P00–P12.

Estadísticas ISTAC

istac_pte_vv.py

PTE vacacional. Dataset C00065A_000061. → tmp/pte_vv_YYYYMMDD.csv. Importar con importar_pte_vv.R.

istac_poblacion_turistica.py

PTE reglada. Dataset C00065A_000042. → tmp/poblacion_turistica_equivalente_YYYYMMDD.csv. Importar con importar_pte_reglada.R.

istac_poblacion.py

Población de derecho (ISTAC). Dataset C00025A_000002. → tmp/poblacion_YYYYMMDD.csv. Importar con importar_poblacion.R.

istac_hogares.py

Hogares y tamaño medio. Dataset C00025A_000001. → tmp/hogares_YYYYMMDD.csv. Importar con importar_hogares.R.

istac_plazas.py

Plazas turísticas regladas históricas. Dataset C00065A_000033. → tmp/plazas_YYYYMMDD.csv. Importar con importar_plazas.R.

istac_turistas.py

Turistas llegados por isla. Dataset E16028B_000011. → tmp/turistas_YYYYMMDD.csv. Importar con importar_turistas.R.

istac_vivienda_construccion.py

Viviendas iniciadas y terminadas. Dataset E25004A_000001. → tmp/vivienda_YYYYMMDD.csv. Importar con importar_vivienda_construccion.R.

Estadísticas INE

ine_poblacion.py

Padrón municipal (INE tabla 29005). Municipios de Canarias, 1996–año actual. → tmp/ine_poblacion_YYYYMMDD.csv. Importar con importar_poblacion_ine.R.

Ventaja: publica el año en curso antes que el ISTAC (datos 2025 disponibles desde enero 2026).

ine_viviendas.py

Censo de viviendas 2021 (INE tabla 59531). → tmp/ine_viviendas_YYYYMMDD.csv. Importar con importar_viviendas.R.

Orden de actualización recomendado

  1. Descargar y procesar registros turísticos: importar_registro_alojamientos.py → P00–P12.
  2. Actualizar PTE vacacional: istac_pte_vv.py → importar_pte_vv.R.
  3. Actualizar población: istac_poblacion.py → importar_poblacion.R, luego ine_poblacion.py → importar_poblacion_ine.R.
  4. Actualizar PTE reglada (anual): istac_poblacion_turistica.py → importar_pte_reglada.R.
  5. Recalcular indicadores: PT01-Capturar_datos_base.R → PT02-Calcular_ratios_dinamicos.R → PT03-Exportar_datos.R.

Logs

Todos los scripts de importación escriben en la tabla de logs de la base de datos mediante helper.R::escribir_log(). Los logs del pipeline P00–P12 también se guardan en ficheros de texto diarios en importar_gobcan/logs/.
