VTP-Canarias — Documentación de la base de datos

Este libro documenta el esquema, el contenido y los procesos de carga de la base de datos viviendas_canarias, que integra registros turísticos, datos estadísticos y cartografía de referencia de Canarias para el cálculo de indicadores de presión turística sobre la población y el territorio.

Estructura del libro

  • Cap. 1 — Visión general y arquitectura: objetivos, stack tecnológico y flujo de datos.
  • Cap. 2 — Pipeline de importación turística: scripts P00–P12, tablas staging_import y alojamientos.
  • Cap. 3 — Tablas geográficas de referencia: islas, municipios, localidades, callejero y centroides.
  • Cap. 4 — Datos estadísticos: población, PTE, hogares, viviendas, superficies, plazas históricas y turistas.
  • Cap. 5 — Clasificación turística: modalidades, tipologías, clasificaciones, destinos turísticos.
  • Cap. 6 — Salida calculada y visualización: snapshots, vista materializada y diccionario de datos.
  • Cap. 7 — Scripts de descarga y mantenimiento: scripts Python de descarga desde CKAN, ISTAC e INE.
